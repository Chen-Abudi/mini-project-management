# ✨ How to Get Started:

## Clone Project:

```bash

git clone https://github.com/Chen-Abudi/mini-project-management.git

```

## Enter the project directory:

```bash

cd mini-project-management

```

## Install the Dependencies:

```bash

npm install

```

or in short

```bash

npm i

```

## Launch:

```bash

npm run dev

```
